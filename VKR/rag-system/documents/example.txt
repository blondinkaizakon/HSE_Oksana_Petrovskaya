Это пример документа для тестирования RAG системы.

RAG (Retrieval-Augmented Generation) - это подход к генерации текста, который объединяет поиск информации из базы знаний с генерацией ответов с помощью языковых моделей.

Основные компоненты RAG системы:
1. Векторное хранилище для хранения документов
2. Модель embeddings для преобразования текста в векторы
3. Система поиска для нахождения релевантных документов
4. Языковая модель для генерации ответов на основе найденной информации

Преимущества RAG:
- Точность: ответы основаны на реальных документах
- Актуальность: можно легко обновлять базу знаний
- Прозрачность: можно увидеть источники информации
- Контроль: можно фильтровать и проверять источники

Этот документ будет использован для демонстрации работы системы поиска.
